## 📋 トラブルシューティング

### ❌ アクセスできない場合

**1. ネットワーク確認**
```bash
# Windows コマンドプロンプトで実行
ping 192.168.255.89
```

**2. ファイアウォール確認**
- Windows Defender > 詳細設定
- ポート8501の受信規則確認

**3. ブラウザ確認**
- キャッシュクリア
- 別ブラウザで試行
- プライベートモードで確認

**4. デバイス確認**
- WiFi接続状況
- 同一ネットワーク接続確認
- VPN切断

### 🔄 システム再起動
```batch
# START_TEAM_MOBILE.batを再実行
1. 既存プロセス終了
2. バッチファイル実行  
3. 新しいアクセスURL確認
```
