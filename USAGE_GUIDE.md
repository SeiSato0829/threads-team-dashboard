# Threads自動管理システム使用ガイド

## 🚀 システム構成
1. **投稿生成システム**: THREADS_ULTIMATE_START.bat
2. **自動投稿実行**: BUZZ_SETUP.bat
3. **管理ダッシュボード**: http://localhost:8501

## 📊 ダッシュボード機能
- **📈 概要**: 投稿統計とKPI
- **📝 履歴**: 全投稿の閲覧・編集
- **📊 分析**: パフォーマンス詳細分析
- **✏️ 編集**: 予定投稿の内容変更
- **👥 共有**: チーム情報共有
- **📋 レポート**: 自動レポート生成

## ⚙️ 自動化設定
- データ更新間隔: 5分
- 自動レポート: 有効
- チーム通知: 有効

## 🔧 トラブルシューティング
1. ダッシュボードが表示されない
   → DASHBOARD_START.bat を実行
2. データが更新されない
   → AUTO_DASHBOARD_UPDATER.bat を確認
3. エラーが発生する
   → ログファイルを確認
